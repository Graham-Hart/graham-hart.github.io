CONWAY'S GAME OF LIFE

INSTRUCTIONS

Start the game by marking cells as "alive" or "dead". Click to mark a cell as alive, shift-click to mark it as dead. Once you're done, press space. Every timestep, cells will re-evaluate their state based on these conditions:


DEAD:

-- If a cell has less than 2 neighbors, it will die.
-- If a cell has greater than 3 neighbors, it will die.


REBORN:

-- If a dead cell has exactly 3 neighbors, it will become alive



If none of these cases are true, the cell will keep it's state.

Keep your cells alive for as long as possible!
